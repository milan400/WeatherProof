## 🎯 Overview

WeatherProof leverages **Gemma 4's multimodal vision-language capabilities** to intelligently detect adverse weather conditions and orchestrate specialized AI restoration models for targeted video enhancement. No cloud required. No GPU dependency. Just clear vision, anywhere.

### 🔍 The Problem
Bad weather blinds cameras. For autonomous vehicles, drones, security systems, and content creators, this means lost data, unsafe operations, and failed missions.

### 💡 Our Solution
Gemma 4 sees each frame, understands the context, and directs specialized models to restore clarity—all running locally on edge devices.
